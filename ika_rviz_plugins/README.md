# ika_rviz_plugins

This repository contains a set of RViz plugins for visualizing different types of data that are not directly associated with a specific repository (e.g. [perception_interfaces](https://github.com/ika-rwth-aachen/perception_interfaces)).

| Plugin | Description | Screenshot |
| --- | --- | --- |
| **Geofence Display** | Visualizes `geometry_msgs/msg/PolygonStamped` messages as geofences in RViz. | ![Geofence Display](assets/geofence.png) |

## Dummy publisher for testing
Dummy publisher scripts are available in [scripts](scripts) for testing the plugins.
